## Chatbot Project


The concept of making a chatbot is pretty new to me. Although I have used chatbots when visiting sites, it did not occur to me to be curious about how one makes it.
Although the concept of chatbots isn’t new to me, the concept of creating one, however, is. I started this project with no idea about what to do. I wasn’t even sure if I should start off on an already existing template or to start from scratch. I searched around the internet for a bit, looking for tutorials on how the website works. Eventually, I figured out how to make a chatbot by trying to mimic the example chatbot given. Initially, I didn’t realise that I was supposed to connect the start button to whichever set of prompts I wanted to display. My chatbot is mainly a simple connection of prompts and buttons. Despite being a very simple chatbot, it is very informative about the programmes being offered at APU.
I also had a hard time figuring out how to add links to the chatbot in case the user wanted to explore a programme even more and check out its website.
Along with that, figuring out how to jump to other components of the chatbot was also challenging, and even confusing. I realised that it is very easy to mix up the connections and end up in a spot where one has to undo all the connections and redo it.
I find this entire process to be pretty similar to coding and algorithms. When a piece of code doesn’t work the way it was intended to, my first instinct is to look back at the code with keen eyes to see if there was something that I missed. If that doesn’t work, I write down the dry run of a code using a sample input. In a very similar fashion, I found myself tracing my path along the flowchart.
I find the use of flowcharts quite useful. It’s a very simple way to follow through a train of thought process, as long as the lines connecting each bubble don't fall on top of each other.
Once I got past the initial difficulties of how the entire thing should work, the rest of the process of putting in information about each programme was easy.
