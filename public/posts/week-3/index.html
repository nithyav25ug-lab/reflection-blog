Week 3

February 28, 2026

Blog Setup

This week was really challenging in comparison to the previous two. I’m grateful that the assignment was broken up into smaller tasks, making it easier to comprehend.

Task 1

Our first task of the week was to create a GitHub account, which was simple.

Task 2

Next, we were supposed to create a SSH key and add it to our Github account that we had previously created. This SSH key is meant to help write into our GitHub repositories using a private key file. The entire concept of SSH keys are pretty foreign to me; so figuring out how to work with them and to connect it took more time than anticipated as I went in clueless about how it works, or what each line I run means, or, even worse, what each error means. Through many sessions troubleshooting, I finally was able to navigate my way through and connect my SSH key to my GitHub account.

Task 3

Now, we put to use the software we downloaded last class: Hugo. It took me a while to figure out which terminal to use. The tutorial given in class mentioned not to use the Command Prompt or Windows PowerShell, but from a Linux terminal. After a bit of reading and researching, I realised that I had to use Git Bash as my terminal. The first step after this is to set up a theme for my blog; I went with the ananke theme.

Task 4

For this task, I faced a bit of an obstacle in creating a GitHub repository and hosting my site on GitHub pages as I had a hard time comprehending the given commands. Although the tutorial given had screenshots of how the page is supposed to look like and where to go, I couldn’t find the buttons being shown. Finally, I managed to find it with a little digging around.

Task 5

It was now time to make my first post. After passing so many obstacles, my mind failed me in trying to come up with any content for my first post. So I decided to go with a simple, one-line introduction to my page.

Once all of this was done, I could finally leave after filling in another survey, just like last week.
